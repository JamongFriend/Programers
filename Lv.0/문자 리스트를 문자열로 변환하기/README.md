[문제링크](https://school.programmers.co.kr/learn/courses/30/lessons/181941)

### 성능 요약
0.02ms, 4.14MB

### 문제 설명
문자들이 담겨있는 배열 arr가 주어집니다. arr의 원소들을 순서대로 이어 붙인 문자열을 return 하는 solution함수를 작성해 주세요.

### 제한사항
-100,000 ≤ a, b ≤ 100,000

### 입출력 예
|arr|result|
|---|---|
|["a","b","c"]|"abc"|
